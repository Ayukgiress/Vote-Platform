## 📖 About

This is the frontend for VotHub — an interactive voting platform that enables users to create polls, vote in real time, and view dynamic results instantly. Built with React and Vite for a fast and seamless experience.

---

## ✨ Features

- 📊 **Create Polls** — Build custom polls in seconds
- ⚡ **Real-time Results** — Watch votes update live as they come in
- 🔐 **User Authentication** — Secure login and signup with protected routes
- 🌍 **Share Polls Publicly** — Share any poll with a link for anyone to vote

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm or yarn

### Installation

**1. Clone the repository:**
```bash
git clone https://github.com/Ayukgiress/Vote-Platform.git
cd Vote-Platform
```

**2. Install dependencies:**
```bash
npm install
```

**3. Set up environment variables:**

Create a `.env` file in the root folder:
```env
VITE_API_URL=http://localhost:5000
```

**4. Start the development server:**
```bash
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.
